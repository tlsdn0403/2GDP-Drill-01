동적 자료형

풍부한 기본 라이브러리 함수

장점: 쉽다, 간결하다, 빠르게 개발 가능
단점: 느리다, 너무 유연하다,진입 장벽이 낮다

//,/ : 나누기 연산자 /은 소수점까지 출력 //은 정수값만 출력,  %:나머지 연산자 , **:제곱 연산자

변수: 값을 저장해놓는 컴퓨터 메모리 안의 공간
mins=' 4 ' 를 int 자료형으로 변환시키려면  m=int(mins)   m은 int자료형임 

슬라이스: 문자열의 일부분을 잘라내는 기법
name[ start : stop : step ]
[0:6] 0번째부터 6번째까지
[:6] 0번째부터 6번쨰
[::2] :2씩 잘라서 출력
[::-1]:-1씩 잘라서 출력 (뒤에서부터 거꾸로 출력)

List
순서가 있는 중복을 허용하는 데이타들의 집합
원하는 데이터를 찾기 위해서 순서 인덱스를 사용
twice[.... , ..... , ....]

Dictionary 
검색을 위한 키를 갖는 데이타들의 집합
{ momo:80 , zwi:85 ,sana:90}

Tuple 
순서가 있는, 중복을 허용하는 데이터들의 집합
데이터 값 변경하는것은 불가능
(val1,val2,-----)

Set
중복을 허용하지 않는 순서에 상관없는 데이타들의 집합
{val1,val2,---}



함수
어떤 특정한 일을 처리하는 기능을 모아놓은 것

게임: 가상월드에 존재하는 여러 개체들의 상호작용
2D게임 : 현재 진행중인 게임을 화면에 2d그림으로 보여주는 것

PNG: 투명한 부분을 집어 넣을 수 있음 (게임만들 때 사용) 무손실 압축
JPG:압축 손실이 일어남


랜덤한 알파벳 뽑기
 a=random.choice("asdafgfqwe)
random.choice 문자열에서 랜덤하게 출력


스프라이트 : 게임 장면 안에서 보여지는 이미지 또는 애니매이션되는 오브젝트

키보드 및 마우스 입력 처리 과정

Step 1: 입력 이벤트들을 폴링한다. (get_events())

Step 2: 이벤트의 종류를 구분한다 (event.type 을 이용)

Step3: 실제 입력값을 구한다 (event.key 또는 event.x , event.y 등 을 이용)
